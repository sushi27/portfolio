- **Dark/Light Mode** *(coming soon)*  

## 🏗 Development Setup  

```bash
# Clone the repository
git clone https://github.com/sushi27/portfolio.git

# Install dependencies
npm install

# Run development server
npm run dev
```  

## 🚀 Deployment  
Deployed on **Vercel** with a **CI/CD pipeline**:  

```bash
# Production build
npm run build

# Deploy to Vercel
vercel --prod
```
